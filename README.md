# Sistema Bancário em Java

Projeto desenvolvido para praticar Programação Orientada a Objetos (POO) em Java.

## Conceitos aplicados
- Classes e métodos abstratos
- Herança
- Polimorfismo
- Interfaces
- Exceções personalizadas
- Encapsulamento
- Coleções (HashMap, ArrayList)
- Boas práticas de modelagem

## Estrutura do Projeto
- application → classe principal (Program)
- entities → entidades do sistema bancário
- exceptions → exceções customizadas
- interfaces → contratos de comportamento

## Funcionalidades
- Criação de contas corrente e poupança
- Transferências entre contas
- Taxa de manutenção
- Rendimento de poupança
- Encerramento de conta com validação
- Relatório geral de contas

## Como executar
1. Clone o repositório
2. Abra o projeto em uma IDE Java (IntelliJ, Eclipse ou VS Code)
3. Execute a classe `Program.java`
